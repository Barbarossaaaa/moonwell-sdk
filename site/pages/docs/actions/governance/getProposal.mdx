# getProposal

Returns the proposal.

## Usage

:::code-group

```ts twoslash [example.ts]
import { moonwellClient } from './client'

const proposal = await moonwellClient.getProposal({ // [!code focus:4]
  proposalId: 1,
  chainId: 1284,
});
```

- **Type:** `number`

The ID of the proposal.


### chainId

- **Type:** `number`
